## Proposed Changes

### 1. 온보딩 모달 (Init Page)
- **[NEW] `index.html` & `_main.js` 업데이트**:
  - `index.html`에 대형 모달 레이아웃(가이드 팝업)을 추가합니다.
  - 설명서 내용(8대 거래소 멀티 토글, 방향키 기능, 퀵뷰 멀티뷰, 김프/펀딩비 통합 제공 등)을 가독성 있게 배치합니다.
  - "다시 보지 않기" 체크박스를 구현하여 `localStorage`에 저장합니다.
  - 설정 모달 내부에 "가이드 다시 보기" 버튼을 추가하여 언제든 열 수 있도록 합니다.

---

### 2. UI 및 UX 개선 사항
- **[MODIFY] 프리셋 제어 UI (`table_filter.js`)**:
  - 기존 항상 보이던 '저장/삭제' 버튼을 숨깁니다.
  - 사용자가 P1~P5 프리셋 중 하나를 클릭해서 활성화했을 때만 해당 프리셋 번호에 대한 '저장/삭제' 버튼이 우측에 페이드인(Fade-in) 되도록 수정합니다.
  - 완료 시 알림창(`alert()`) 대신 **SweetAlert2 토스트 팝업**을 사용하여 우측 하단에 조용하고 이쁜 형태로 피드백을 줍니다.
  
- **[MODIFY] 즐겨찾기 코인 개수 표시 (`table_render.js` & `table_filter.js`)**:
  - 탭 렌더링 시, 로컬 스토리지에 저장된 FAV 및 FAV2 배열의 길이를 읽어와서 버튼 텍스트 옆에 `(N)` 형태로 표기합니다.

- **[MODIFY] 검색창 위치 및 로직 변경 (`index.html`, `ui_control.js`, `table_filter.js`)**:
  - 검색창을 사이드바 하단이나 외부에서 **좌측 테이블 패널 최상단(탭 위)**으로 이동시킵니다.
  - 검색 시 드롭다운 팝업을 띄우는 기존 로직 대신, 입력하는 즉시 테이블 리스트 자체가 필터링되도록 `store.searchQuery` 전역 상태를 도입합니다.
  - `getFilteredData()`에서 검색어를 기반으로 테이블 목록을 즉각 렌더링합니다.

- **[MODIFY] MCap, VMC 니은자(L자) 배경 효과 (`table_render.js`)**:
  - `col-vol-b` (바이낸스 볼륨/시총) 셀에 좌측+하단 테두리/그라데이션 배경(노란색 계열)을 줍니다.
  - `col-vol-u` (업비트 볼륨/VMC) 셀에는 우측+하단(또는 좌우 반전 L자 형태) 그라데이션 배경(파란색 계열)을 적용해 시각적 묶음 효과를 강조합니다.

### 3. 차트 및 그리기 도구 관련 개선
- **[MODIFY] 그리기 도구 제거 및 자(Ruler) 측정 도구 분리 (`chart.js` / `index.html`)**:
  - 기존 우측 또는 좌측에 배치했던 버그가 많은 그리기 도구 툴바를 완전히 제거합니다.
  - '자(Ruler) 측정 기능' 하나만 살려내어, 차트 상단 컨트롤 바에 단독 버튼으로 배치합니다.

- **[MODIFY] AL (Auto/Log) 버튼 위치 및 스케일 겹침 수정 (`ui_control.js` / CSS)**:
  - 차트 오버레이 영역에 있는 A, L 버튼 컨테이너를 조금 더 좌측으로 이동(예: `right-16` -> `right-20` 등)하여 가격 스케일 숫자와 겹치지 않도록 수정합니다.

- **[MODIFY] Autofit 동작 시 3개 패널 (Chart, Vol, Kimchi) Width 싱크 불일치 버그 (`chart.js`)**:
  - A 버튼(Autofit)을 누르면 가격 스케일 너비가 자동 조절되는데, 이때 하단의 거래량, 김프 패널의 스케일 너비가 따로 노는 버그를 잡습니다. `resetPriceScaleWidthSync()`를 강제 호출하여 3개의 Pane 레이아웃이 완벽히 정렬되도록 합니다.

- **[MODIFY] 로딩 완료 시 Autofit 자동 발사 (`chart_data.js`)**:
  - 차트 히스토리(캔들, 거래량, 김프) 패치가 완전히 끝나는 `Promise.all` 블록 마지막 부분에서 모든 시리즈에 대해 `.timeScale().fitContent()`를 한 번 트리거해 줍니다.

- **[MODIFY] Pure Chart 시작 시 효과 누락 버그 (`z_style.css`, `_main.js`)**:
  - 앱 진입 시 Pure chart 탭이 활성화된 상태라면, 헤더와 불필요한 컨트롤이 즉각 숨겨지는 CSS / JS 상태 동기화 누락을 수정합니다.

### 4. 김프 데이터 / 실시간 렌더링 버그 수정
- **[MODIFY] 김프 빗썸 디폴트 및 실시간 출렁임 문제 (`chart_data.js`, `stream_korea.js`)**:
  - 김프를 계산할 때 한국 거래소 우선순위를 '빗썸'에서 '업비트'로 1순위 상향합니다.
  - 빗썸 과거 김프 캔들 수치가 이상하게 나오는 부분의 API 처리 로직을 수정합니다.
  - 실시간 스트리밍 시, 메인 차트 캔들만 업데이트할 게 아니라, 한국 거래소 가격 틱이 들어올 때마다 **마지막 김프 캔들(Line/Histogram)** 값을 기존 값에 덮어씌우는 방식으로 수정하여, 가격이 튀는(출렁이는) 버그를 방지합니다.

### 5. 신규 기능: 가격 알람 (DB 없음) 및 퀵뷰 프리셋
- **[NEW] 로컬 가격 알람 기능 (`stream_render.js`, `index.html`)**:
  - 코인명 우측 혹은 툴바에 종 모양 🔔 아이콘 버튼을 추가합니다.
  - 클릭 시 모달창에서 '목표가'를 설정하면 `localStorage`에 알람을 등록합니다.
  - 실시간 소켓에서 틱 데이터가 들어올 때, 현재 가격이 알람 가격을 돌파했는지 체크하고 도달 시 Toast 알림을 발생시킵니다.
  
- **[MODIFY] 퀵뷰 거래소 멀티 토글 및 프리셋 무제한 확장 (`quickview.js`)**:
  - 퀵뷰 팝업 내부 상단에도 기존 메인 테이블에 있는 '8대 거래소 멀티 토글' 바를 주입합니다.
  - 메인 테이블 5개 제한이 아닌, 배열 기반으로 `+` 버튼을 눌러 개수를 제한 없이 늘릴 수 있는 무제한 프리셋 슬롯을 구현합니다.
