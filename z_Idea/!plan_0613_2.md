# 테이블 0초 데이터 바인딩 정렬 전환 계획

물리적으로 HTML 노드(DOM)의 배치 순서를 계속 뒤섞는 무거운 연산(`insertBefore`)을 완전히 없애고, 800개의 고정된 테이블 행에 정렬된 데이터를 논리적으로 매핑하여 텍스트 및 지표만 빠르게 교체하는 데이터 바인딩 방식으로 전환하여 즉각적인 정렬을 달성합니다.

## 사용자 검토 필요 사항

> [!IMPORTANT]
> - 기존에 적용되어 있던 스르륵 움직이는 정렬 애니메이션(FLIP)은 물리적인 노드 이동이 일어나지 않으므로 비활성화(제거)됩니다.
> - 대신, 정렬 헤더를 클릭하는 순간 화면 떨림이나 밀림 없이 0.01초 만에 수치와 텍스트가 순식간에 교체되는 극대화된 반응 속도를 확보하게 됩니다.

## 제안된 변경 사항

### 테이블 렌더링 컴포넌트

#### [수정] [table_render.js](file:///c:/Users/kmj/Sellnance/static/table_render.js)

- `renderTable(isRealtime)` 함수 내부에서 `tbody.insertBefore`를 호출해 HTML 노드 순서를 뒤바꾸던 무거운 정렬 연산을 제거합니다.
- 800개의 테이블 행(`.coin-row`)의 순서는 그대로 유지한 채, 정렬/필터링이 완료된 `store.currentTableData` 배열 순서에 맞춰 각 HTML 행에 데이터만 대입합니다.
- 실시간 가격 웹소켓 데이터가 수신될 때 올바른 행을 찾을 수 있도록, `rowEl.dataset.sym` 및 `store.rowDomMap` 매핑 정보를 매번 정확하게 업데이트합니다.

```javascript
// 변경 예정인 renderTable 함수의 핵심 구조
export function renderTable(isRealtime = false) {
  const tbody = document.getElementById("coin-list-body");
  if (!tbody) return;

  const filteredData = getFilteredData();
  const totalCount = filteredData.length;

  // 1. DOM 개수가 일치하지 않을 때만 최초 1회 껍데기 생성
  if (!store.tablePoolInitialized || tbody.children.length !== totalCount) {
    // 최초 껍데기 풀 생성 로직 실행
    ...
  }

  // 2. DOM 노드 이동을 완전히 차단하고 O(N)으로 텍스트 및 지표 데이터만 교체
  store.visibleSymbols.clear();
  store.rowDomMap = new Map(); // 실시간 웹소켓 가격 업데이트용 매핑 최신화

  for (let i = 0; i < totalCount; i++) {
    const rowEl = tbody.children[i];
    const rowData = filteredData[i];
    
    if (rowEl && rowData) {
      // 실시간 웹소켓 매핑 정보 업데이트
      rowEl.dataset.sym = rowData.Ticker;
      store.rowDomMap.set(rowData.Ticker, rowEl);
      
      const isPending = ...;
      const needsStatic = ...;
      if (needsStatic) {
        updateRowStaticHTML(rowEl, rowData);
      }
      
      // 상위 30위 또는 가시 영역만 즉각 동적 수치 업데이트
      if (i < 30) {
        store.visibleSymbols.add(rowData.Ticker);
        updateRowDynamicHTML(rowEl, rowData);
      }
    }
  }

  updateBoundaryClass(tbody);
  applySelectedHighlight();
}
```

### 수동 검증
- 가격, 거래량, 변동률 헤더를 클릭했을 때 화면 떨림이나 로딩 대기 현상 없이 즉시 수치가 정렬되는지 확인.
- 실시간 가격 및 김프/펀딩비가 업데이트될 때 꼬임 없이 해당 코인의 행에 정상적으로 출력되는지 확인.
