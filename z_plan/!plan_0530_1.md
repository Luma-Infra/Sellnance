# 계획: 바이낸스 주식/파생상품 코인 분류 및 필터링 (바낸 필터 3버튼 분리)

바이낸스 토큰화 주식 및 주식 파생 코인들(예: AAPL, TSLA, NVDA, AMZN 등)을 일반 암호화폐 목록에서 분리하여 가시성을 높이고 마켓을 깔끔하게 유지하는 기능 구현 계획입니다.

상단 다중 거래소 필터바에서 **BINANCE 관련 필터를 단일 버튼의 순환 토글 방식 대신 B-SPOT, B-FUT, B-STOCK 3개의 개별 버튼으로 분리**하여, 여러 번 누르지 않고 직관적으로 제어할 수 있도록 개선합니다. 또한, 기본 `MAIN LIST` 탭 조회 시 주식 및 파생 토큰들은 기본 노출 대상에서 제외하여 목록을 깔끔하게 유지합니다.

## 제안된 변경 사항

### 프론트엔드 코드

#### [수정] [table_filter.js](file:///c:/Users/kmj/Sellnance/static/table_filter.js)
- 코인의 영문명(`Name`)에 "stock" 또는 "derivative"가 포함되어 있는지 식별하는 헬퍼 함수 `isStockCoin(row)`를 추가합니다. (단, 일반 크립토 프로젝트인 `Rootstock (RIF)` 등은 제외하도록 예외 처리 적용)
- `getFilteredData()` 필터링 로직 수정:
  - 기본적으로 주식 토큰(`isStockCoin(row) === true`)은 메인 리스트에서 제외 처리합니다.
  - 단, 즐겨찾기(`FAV`, `FAV2`) 탭에서는 주식 토큰 여부와 상관없이 모두 보여줍니다.
- **다중 거래소 필터 상태 필터링 개편**:
  - `store.exchFilterStates`에 바이낸스 관련 상태를 기존 1개(`BINANCE`)에서 3개로 분할하여 관리합니다:
    - `BINANCE`: 바이낸스 현물 (B-SPOT)
    - `BINANCE_FUTURES`: 바이낸스 선물 (B-FUT)
    - `BINANCE_STOCK`: 바이낸스 주식 (B-STOCK)
  - `getFilteredData()` 내부의 포함/제외 필터 판별 로직을 분리된 바이낸스 3종 키 값에 매칭되도록 업데이트합니다.
  - B-STOCK 필터가 활성화(state === 1)되었을 때만 주식 코인들이 필터링 목록에 포함되어 보이도록 설계합니다.
  - 상단 필터바 UI 렌더링 함수(`updateExchFilterUI()`)를 수정하여 바이낸스 자리에 **B-SPOT, B-FUT, B-STOCK** 3개의 독립된 버튼을 렌더링하고, 각 버튼의 상태 순환(0: 미선택 -> 1: 포함 -> -1: 제외 -> 0)을 개별 동작하도록 연결합니다.

## 검증 계획

### 수동 검증
1. 로컬 개발 서버 웹앱 대시보드 접속.
2. `MAIN LIST` 탭 선택: 기본 상태에서 TSLA, AAPL, AMZN 등의 주식 코인이 목록에 나타나지 않는지 확인.
3. 상단 필터바에서 **B-STOCK** 필터를 활성화(초록색)하고 목록에 주식 토큰만 정상 노출되는지 확인.
4. 상단 필터바에서 **B-SPOT, B-FUT, B-STOCK** 개별 버튼을 토글하며 즉각적인 필터링(포함/제외/해제)이 직관적이고 빠르게 동작하는지 검증.
