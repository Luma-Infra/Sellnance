# 전체화면 툴바 및 레이아웃 복원 버그 수정 계획

차트 전체화면(🖥️) 진입 시 타임프레임 옆 툴바 영역에 필요한 버튼들(화면 복원, 자, 우측 퍼센트, OHLC, 카운트다운)을 배치하고 호가창 열기 버튼은 제외하며, 전체화면을 끌 때 버튼들의 원래 위치가 깨지는 버그를 해결합니다.

## 제안된 변경 사항

### UI 제어 로직 수정

#### [MODIFY] [ui_control.js](file:///c:/Users/78831/Sellnance/static/ui_control.js)
- 전체화면 토글 로직 리팩토링:
  - `#tf-container` (타임프레임 컨테이너), `#head-control-buttons` (컨트롤 버튼 묶음), `#chart-fullscreen-btn` (전체화면 버튼)의 **원래 부모 노드(Parent)**와 **다음 형제 노드(Next Sibling)** 위치를 정확히 변수에 기록합니다.
  - 전체화면 진입 시:
    - `#tf-container`를 차트 영역인 `#chart-wrapper` 내부의 최상단으로 이동합니다.
    - `#head-control-buttons`를 `#tf-container` 안으로 이동시키고 `margin-left: auto`를 주어 우측 정렬합니다.
    - `#toggle-orderbook-btn` (호가창 열기) 버튼은 스타일 처리를 통해 숨깁니다.
    - 전체화면 취소 버튼(화면 복원)을 `#tf-container` 내부의 가장 우측 끝에 배치합니다.
  - 전체화면 탈출 시:
    - 기록해 둔 원래 부모와 다음 형제 노드 정보를 사용하여 `#tf-container`, `#head-control-buttons`, `#chart-fullscreen-btn`을 원래의 위치와 순서대로 완벽하게 복원합니다. (`insertBefore` 활용)
    - 우측 정렬이나 마진 등의 임시 인라인 스타일을 제거합니다.

## 검증 계획

### 수동 검증
1. 차트 영역 우측 상단의 **🖥️ 전체화면** 버튼을 클릭합니다.
2. 전체화면 모드에서 차트 최상단 툴바에 타임프레임 버튼과 함께 **자(Ruler)**, **우측 퍼센트% 끄기/켜기**, **OHLC 끄기/켜기**, **카운트다운**, **화면 복원** 버튼이 나란히 배치되어 있는지 확인합니다.
3. **호가창 열기** 버튼은 보이지 않는지 확인합니다.
4. **화면 복원** 버튼을 클릭하여 전체화면을 탈출합니다.
5. 원래 화면으로 돌아왔을 때, 컨트롤 버튼들과 타임프레임 영역이 깨지거나 밀리지 않고 원래 레이아웃 위치에 깔끔하게 복구되는지 확인합니다.
6. 빌드 명령어 `npx vite build` 등을 수행하여 프로덕션 빌드에서도 정상 동작하는지 테스트합니다.
