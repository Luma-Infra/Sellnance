# Sellnance 주요 업데이트 요약

## 1. 온보딩 모달 (초기 환영 가이드)
Sellnance 대시보드에 처음 방문하는 사용자를 위한 온보딩 모달(가이드 팝업)을 새롭게 추가했습니다.

- **기능 소개**: 다중 거래소 통합 지원, 퀵뷰 멀티 차트, 로컬 가격 알람, 단축키 지원 등 핵심 기능에 대한 직관적인 안내를 제공합니다.
- **다시 보지 않기**: '다시 보지 않기' 체크박스를 클릭하면 로컬 스토리지에 저장되어 이후 접속 시 모달이 다시 뜨지 않습니다.
- **다시 열기 지원**: 언제든 설정(⚙️) 메뉴 내부의 버튼을 통해 온보딩 가이드를 다시 열어볼 수 있도록 구현했습니다.

## 2. 로컬 가격 알람 (🔔)
백엔드 서버에 부하를 주지 않으면서 브라우저 로컬 환경만으로 동작하는 가격 알람 기능을 추가했습니다.

- **설정 방법**: 메인 테이블에 위치한 코인 이름 옆의 `🔔` 아이콘을 클릭합니다.
- **동작 원리**: 알람을 설정하고자 하는 목표가를 입력받은 뒤, `localStorage`에 안전하게 저장합니다.
- **크로스(교차) 감지**: `stream.js`가 웹소켓 실시간 가격 데이터를 처리하면서, 저장된 목표가를 위 아래로 돌파(Crossover)하는 순간을 즉각 감지합니다. 조건 만족 시 알림(`alert`)을 띄우고 알람을 자동 해제합니다.

## 3. 퀵뷰(QuickView) 무제한 페이지네이션 적용
기존에 32개 코인(4페이지)으로 제한되어 있던 하드코딩 요소를 제거하고, 페이지네이션 한계를 무제한으로 확장했습니다.

- **동적 계산**: 검색 및 필터 상태에 따라 남은 코인의 총 개수를 기반으로 전체 페이지 수(Max Page)가 실시간 동적 계산되어 적용됩니다.

## 4. 자(Ruler) 측정 도구 독립 제어 개선
기존 트레이딩뷰 스타일의 무거운 드로잉 툴바 코드를 완전히 제거한 후, 단독으로 빠져나온 자(Ruler) 도구 코드를 전역(`window.toggleMeasureTool`)으로 안전하게 바인딩하여 Lightweight Charts 네이티브 툴과 완벽하게 맞물려 작동하도록 안정성을 높였습니다.
