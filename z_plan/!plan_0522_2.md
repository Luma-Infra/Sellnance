# 🛠 좌측 테이블 3단 뷰 토글 및 동적 레이아웃 조절 계획

사용자의 편의와 공간 효율성을 극대화하기 위해, 좌측 마켓 리스트(테이블) 영역을 3가지 모드(심플, 베이직, 전문가)로 토글할 수 있는 버튼을 추가하고 레이아웃을 동적으로 반응시킵니다.

## ⚠️ User Review Required

> [!WARNING]
> **패널 너비 동적 변경 및 테이블 열(Column) CSS 제어**
> 토글 버튼 클릭 시 `#left-panel`의 너비(width)가 즉각적으로 변하며, 메인 차트와 우측 호가창의 너비도 이에 맞춰 유연하게 반응(flex-grow)하도록 재설계합니다. 
> - **심플 (Simple)**: 코인 이름(Asset)과 가격(Price)만 표시. (최소한의 너비 차지, 약 260px)
> - **베이직 (Basic)**: 시총(M.Cap), 김프(Kimchi), 상장거래소(Exch)까지 표시. (기존 너비, 약 400px)
> - **전문가 (Expert)**: 베이직 정보 + **최초 상장일(Listing Date)** 표시. (최대 너비, 약 500px)

## ❓ Open Questions

> [!IMPORTANT]
> 1. **상장일 데이터 부재**: 현재 백엔드(파이썬) 구조상 바이낸스 선물(Futures)에는 `onboardDate`(상장일)가 API로 제공되지만, 현물(Spot)이나 업비트 등에는 상장일 데이터가 기본 제공되지 않습니다. 선물이 아닌 코인들은 빈칸(`-`)으로 둘까요?
> 2. **3단 토글 UI 위치**: 검색바(Search) 바로 밑, 즉 테이블 헤더(Asset, Price 등) 바로 위에 버튼 3개가 나란히 있는 작은 탭(Tab) 형태로 넣는 것이 가장 예쁠 것 같은데 동의하시나요?

## Proposed Changes

### UI 레이아웃 및 뼈대 DOM 설계

`templates/index.html` 내 좌측 패널을 수정하여 3단 토글 버튼을 삽입하고, 모드별 CSS 클래스를 부여합니다.

#### [MODIFY] [index.html](file:///c:/Users/78831/Sellnance/templates/index.html)
- `#left-panel` 내 검색바 하단 영역에 3개의 버튼(`[ 심플 | 베이직 | 전문가 ]`)으로 구성된 토글 그룹 추가.
- 전역 CSS 클래스를 컨트롤하기 위해 버튼 클릭 시 `#left-panel` 요소에 `view-mode-simple`, `view-mode-basic`, `view-mode-expert` 클래스 토글 로직(`ui_control.js` 호출) 적용.

### 테이블 렌더링 및 CSS 제어

HTML 테이블 열(Column) 렌더링을 담당하는 CSS와 스크립트를 업데이트합니다.

#### [MODIFY] [z_style.min.css](file:///c:/Users/78831/Sellnance/static/z_style.min.css) (or z_style.css)
- 부모 요소가 `view-mode-simple`일 때 `.col-mcap`, `.col-kimch`, `.col-exch` 등 불필요한 열에 `display: none` 적용 및 `#left-panel` 너비를 `260px`로 축소.
- `view-mode-expert`일 때 새롭게 추가할 `.col-listing` 표시 및 너비를 `500px`로 확장.
- CSS 전환(transition) 효과를 추가하여 너비가 부드럽게 쌀먹 애니메이션으로 줄어들고 늘어나게 처리.

#### [MODIFY] [table_render.js](file:///c:/Users/78831/Sellnance/static/table_render.js)

#### [MODIFY] [ui_control.js](file:///c:/Users/78831/Sellnance/static/ui_control.js) & [_store.js](file:///c:/Users/78831/Sellnance/static/_store.js)
