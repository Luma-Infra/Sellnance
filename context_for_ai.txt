# 🚀 HTS 쌀먹 최적화 호가창(Orderbook) 적용 완료!
### 🌟 주요 구현 포인트
### 🛠 세부 작동 원리
1. **마켓 자동 동기화**: 업비트 차트를 보고 있으면 업비트 원화 호가창이, 바이낸스 선물을 보고 있으면 바이낸스 선물 100ms 호가창이 정확히 실시간으로 매칭되어 표기됩니다.
2. **동적 Precision (+/- 버튼)**: 차트 코인의 기본 소수점(Precision)을 바탕으로 그룹화가 진행되며, 언제든지 사용자가 수동으로 그룹화 틱 사이즈를 늘리거나 줄일 수 있도록 헤더에 오버라이드 기능
을 연결해 두었습니다.
3. **토글 버튼**: 상단 헤더에 `호가창 열기` 버튼을 부착하여 호가창을 부드럽게 열고 닫을 수 있습니다. 패널이 열고 닫힐 때 차트 사이즈도 동기화되어 레이아웃 붕괴를 방지합니다.
4. **테일윈드 테마 연동**: 백그라운드 뎁스 바(Depth Bar)는 `bg-theme-down/15` 및 `bg-theme-up/15` 같은 변수를 사용하여 라이트/다크 모드 상관없이 100% 매끄럽게 호환됩니다. 
(Tailwind JS 빌드 처리 완료)
